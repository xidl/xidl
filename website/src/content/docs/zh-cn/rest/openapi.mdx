`openapi` 目标会为 HTTP 风格接口生成 `openapi_{filename}.json`。

## 最小生成命令

```bash
xidlc gen --out-dir generated openapi api.idl
```

## 当前会读取的 pragma

```idl
#pragma xidlc package "Smart City Public APIs"
#pragma xidlc version "v2.0.0"
```

效果：

- `package` -> `info.title`
- `version` -> `info.version`

如果不写，生成器会回退到默认元数据。
