## 枚举

```idl
enum Color {
    Red,
    Green,
    Blue
};
```

## 结构体

```idl
struct Point {
    uint32 x;
    uint32 y;
};
```

## 接口

```idl
interface ILogger {
    void log(string message);
};
```

## 注解

```idl
struct Point {
    uint32 x;
    @optional
    uint32 y;
};
```

## 容器

```idl
struct Container {
    sequence<string> names;
    map<string, uint32> name_to_id;
};
```

## Union

```idl
enum Color {
    Red,
    Green,
    Blue
};

union ColorPoint switch(Color) {
    case Red:
        uint32 red_x;
        uint32 red_y;
    case Green:
        uint32 green_x;
        uint32 green_y;
    case Blue:
        uint32 blue_x;
        uint32 blue_y;
    default:
        uint32 default_x;
        uint32 default_y;
};
```
