## `@name`

`@name("...")` 用来重命名字段的序列化名字。

- 作用于 `struct` / `union` 成员
- 生成 HTTP / JSON 映射时会把它当作 wire / property name
- 生成 Rust 时通常映射为 `#[serde(rename = \"...\")]`

示例：

```idl
module demo {
  struct User {
    uint32 id;
    @name("full_name")
    string name;
  };
};
```
