---

`xidlc` 在标准 IDL 预处理器指令之外，支持一组控制生成行为的
`#pragma xidlc ...` 扩展。

## 语法

```txt
#pragma xidlc ...
```

## 常见 pragma

### 序列化版本

```txt
#pragma xidlc XCDR1
#pragma xidlc XCDR2
```

也支持：

```txt
#pragma xidlc serialize(XCDR1)
#pragma xidlc serialize(XCDR2)
```

### 显式序列化种类

```txt
#pragma xidlc serialize(CDR)
#pragma xidlc serialize(PLAIN_CDR)
#pragma xidlc serialize(PL_CDR)
```

### OpenAPI 元数据

```txt
#pragma xidlc package "rev-tunnel"
#pragma xidlc version "1.0.0"
```

## 作用域与覆盖

- 同类 pragma 按文件顺序生效，后写覆盖前写
- `package` / `version` 只影响 `openapi`
- `XCDR1`、`XCDR2` 与 `serialize(...)` 影响序列化配置
