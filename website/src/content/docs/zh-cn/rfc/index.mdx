# RFC

本节提供的是规范性说明，而不是入门教程。

如果你只是第一次上手，请先看[指南](../guide/)或[详细文档](../docs/)。

## 本节包含什么

- [HTTP RFC](./http/)
- [HTTP 安全 RFC](./http-security/)
- [HTTP Stream RFC](./http-stream/)
- [JSON-RPC RFC](./jsonrpc/)
- [JSON-RPC Stream RFC](./jsonrpc-stream/)
- [XCDR RFC](./xcdr/)

## 什么时候应该看 RFC

- 你需要精确确认某个映射规则
- 你要实现或修改生成器
- 你在做运行时兼容性或测试矩阵
- 用户文档的描述已经不够精确

## 使用建议

- 先用 Guide 理解主流程
- 再用 Docs 理解日常使用方式
- 最后用 RFC 锁定具体语义
