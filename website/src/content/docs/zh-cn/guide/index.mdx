## 1. 安装 `xidlc`

<Tabs>
  <TabItem label="macOS / Linux">

```bash
curl --proto '=https' --tlsv1.2 -sSf https://xidl.github.io/xidl/public/install.sh | sh
```

  </TabItem>
  <TabItem label="Windows PowerShell">

```bash
iwr -useb https://xidl.github.io/xidl/public/install.ps1 | iex
```

  </TabItem>
  <TabItem label="Cargo">

```bash
cargo install xidlc
```

  </TabItem>
</Tabs>
