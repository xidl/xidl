description: "用一份最小 IDL 跑通 XIDL 的 HTTP 主线。"
---

在安装了 XIDL 后，我们现在看看如何创建一个 HTTP 服务端和客户端代码：

## 目标

我们要完成四件事：

1. 写一个带 HTTP 注解的接口
2. 生成 `rust-axum` 代码
3. 生成 `openapi.json`

## 1. 定义一个 HTTP 接口

创建 `hello_world.idl`：

```txt
interface HelloWorld {
    @post(path = "/hello")
    void sayHello(
        in string name
    );
};
```

## 2. 生成 Axum 代码

```bash
xidlc gen rust-axum hello_world.idl
```

现在 xidlc 会在 `out` 目录下生成服务端代码。
如果你还需要客户端代码，使用：

```bash
xidlc gen rust-axum --client hello_world.idl
```

## 3. 创建服务端

```rust
mod hello_world;

struct HelloWorldImpl;

#[async_trait::async_trait]
impl HelloWorld for HelloWorldImpl {
    async fn sayHello(&self, name: String) -> Result<(), xidl_rust_axum::Error> {
        println!("Hello, {}!", name);
        Ok(())
    }
}

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let addr = "127.0.0.1:3000";
    println!("axum hello_world server listening on {addr}");

    xidl_rust_axum::Server::builder()
        .with_service(HelloWorldServer::new(HelloWorldImpl))
        .serve(addr)
        .await?;

    Ok(())
}
```

## 4. 使用客户端

```rust
#[tokio::main]
async fn main() {
    let client = HelloWorldClient::new("http://127.0.0.1:8080");
    let _ = client.sayHello("world".to_string()).await;
}

```

## 5. 生成 OpenAPI 文档

```bash
xidlc gen openapi hello_world.idl
```
