## 为什么选择 XIDL？

<CardGrid stagger>
	<Card title="跨语言支持" icon="setting">
		一次定义接口，自动生成 Rust、Go、Python 等多种语言的客户端与服务端代码。
	</Card>
	<Card title="多协议兼容" icon="add-document">
		原生支持 HTTP (REST) 和 JSON-RPC，轻松应对不同的业务场景。
	</Card>
	<Card title="强类型安全" icon="pencil">
		基于强类型系统，在编译期拦截错误，提升开发效率与系统稳定性。
	</Card>
	<Card title="现代工具链" icon="rocket">
		配套完整的编译器 `xidlc` 与构建系统，深度集成现有开发工作流。
	</Card>

## 你可以构建什么？

- **微服务通信**：在不同语言编写的服务之间建立可靠的连接。
- **Web API**：快速生成符合规范的 HTTP 接口。
- **实时流**：利用 XIDL 对流式数据的原生支持，构建实时应用。
