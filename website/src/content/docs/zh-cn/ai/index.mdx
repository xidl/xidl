## 常用命令

```bash
xidlc gen --out-dir out rust api.idl
xidlc gen --out-dir out rust-axum api.idl
xidlc gen --out-dir out openapi api.idl
```

## 关键建议

- 优先相信实现支撑的结论
- 结合 `xidlc-examples/` 交叉验证
- 传输语义不清楚时以 RFC 为准
- 目标能力不明确时直接看生成器模块
