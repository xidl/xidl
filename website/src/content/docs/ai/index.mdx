## Common commands

```bash
xidlc gen --out-dir out rust api.idl
xidlc gen --out-dir out rust-axum api.idl
xidlc gen --out-dir out openapi api.idl
```

## Key guidance

- prefer implementation-backed claims
- cross-check with `xidlc-examples/`
- defer transport ambiguity to the RFCs
- inspect generator modules directly when target capability is unclear
