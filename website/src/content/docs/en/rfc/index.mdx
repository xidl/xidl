# RFC

This section provides normative specifications rather than introductory tutorials.

If you are getting started for the first time, please refer to the [Guide](../guide/) or [Detailed Documentation](../docs/).

## What's Included

- [HTTP RFC](./http/)
- [HTTP Security RFC](./http-security/)
- [HTTP Stream RFC](./http-stream/)
- [JSON-RPC RFC](./jsonrpc/)
- [JSON-RPC Stream RFC](./jsonrpc-stream/)

## When to Consult the RFC

- You need to confirm a specific mapping rule precisely.
- You are implementing or modifying a generator.
- You are working on runtime compatibility or a test matrix.
- The descriptions in the user documentation are no longer precise enough.

## Usage Recommendations

- Use the **Guide** to understand the main workflow.
- Use the **Docs** to understand daily usage.
- Use the **RFC** to lock down specific semantics.
