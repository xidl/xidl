XIDL provides an editor integration through the
[IDL language server](https://github.com/xidl/idl-language-server).

It works best in Visual Studio Code, and the same language-server protocol can
also be consumed by other editors.

[![VS Code Extension](https://img.shields.io/badge/Install-VS%20Code%20Extension-007ACC?logo=visualstudiocode&logoColor=white)](https://marketplace.visualstudio.com/items?itemName=cathaysia.vscode-idl-language)

## Features

- syntax highlighting
- formatting
- HTTP API preview
