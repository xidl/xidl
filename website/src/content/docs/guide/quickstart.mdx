## Install `xidlc`

### Release installer

macOS and Linux:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://xidl.github.io/xidl/public/install.sh | sh
```

Windows PowerShell:

```powershell
iwr -useb https://xidl.github.io/xidl/public/install.ps1 | iex
```

### Cargo

```bash
cargo install xidlc
```

### Cargo Binstall

```bash
cargo binstall xidlc
```

## Verify the install

```bash
xidlc --help
```

## Generate a first artifact

```bash
xidlc gen -o out rust your.idl
```
