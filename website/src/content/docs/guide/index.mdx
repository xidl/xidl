# Guide

This section is the shortest path from zero to a working XIDL project.

## Recommended order

1. Read [Install & Quickstart](./quickstart/).
2. Continue with [First HTTP API](./first-http-api/).
3. Set up editor support in [Editor](./editor/).
4. Use the [Docs](../docs/) section for syntax, generators, and references.

## What this section covers

- installing `xidlc`
- writing a minimal `.idl` file
- generating your first target output
- understanding where HTTP and Rust fit into the toolchain

## When to leave this section

Move on to the detailed docs once you need exact target behavior, annotations,
or RFC-level transport rules.
