## Why XIDL?

<CardGrid stagger>
	<Card title="Cross-language" icon="setting">

## What can you build?

- **Microservices**: Reliable connections between services in different languages.
- **Web APIs**: Fast, spec-compliant HTTP interfaces.
- **Real-time Streams**: Built-in support for streaming data.
