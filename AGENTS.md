# AGENTS

1. The file must not exceed 300 lines.
2. Short unit tests should live in the module's `test.rs` module.
3. Multiple unit tests should live under the `mod/tests/` directory.
4. Never mix source code and unit tests in the same file.
5. Follow the `code-style` skill in `.agents/skills/code-style/` before
   finishing any change or creating a commit.
